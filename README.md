# CMB Power Spectrum Analysis using Planck 2018 Data

## Overview
This project presents a computational pipeline for reconstructing the Cosmic Microwave Background (CMB) angular power spectrum using Planck 2018 SMICA temperature maps.

The analysis includes:
- HEALPix-based map preprocessing
- Galactic masking
- Spherical harmonic decomposition
- Angular power spectrum reconstruction
- Acoustic peak analysis
- Comparison with ΛCDM theoretical predictions using CAMB
- Simplified MCMC-based cosmological parameter estimation

## Key Results
- Acoustic peaks detected at ℓ ≈ 220, 540, and 815
- Strong consistency with ΛCDM cosmology
- RMS residual ≈ 4.2%
- Reproducible computational workflow
